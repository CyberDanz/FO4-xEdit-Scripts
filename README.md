# FO4 xEdit Scripts

xEdit (FO4Edit) scripts for Fallout 4 modding, by CyberDanz.

## Scripts

- **[Cyber's Leveled List Auditor v4](CybersLeveledListAuditV4/)** — audits leveled lists for structural problems, balance outliers, and injected entries lost to load order conflicts.

## Installation

Each script folder contains its own README. Generally: copy the `.pas` file to `<xEdit folder>\Edit Scripts\`, then right-click a plugin in xEdit and choose Apply Script.
